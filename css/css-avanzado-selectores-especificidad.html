CSS Avanzado: Selectores y Especificidad

¿Qué son los selectores?

Los selectores en CSS sirven para elegir a qué elementos HTML quieres aplicar estilos. Algunos ejemplos sencillos:

  • Selector de etiqueta: p (selecciona todos los párrafos)
  • Selector de clase: .importante (selecciona los elementos con class="importante")
  • Selector de id: #menu (selecciona el elemento con id="menu")
