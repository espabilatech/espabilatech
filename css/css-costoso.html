Cambios CSS más costosos

¿Qué cambios de CSS afectan más al rendimiento?

  • Propiedades que disparan reflow (layout):
    • width, height, margin, padding, top, left, bottom, right
    • position, display, float, clear
    • font-size, font-family
    • border-width, min/max-width, min/max-height
  • Propiedades que disparan repaint:
    • color, background, border-color, box-shadow, visibility, outline
  • Propiedades que disparan compositing (nueva layer):
    • transform, opacity, filter, will-change

Ejemplo interactivo

Prueba cambiar margin vs background en la consola y observa el impacto en el renderizado.
