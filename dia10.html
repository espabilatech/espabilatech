Proyecto final

Qué aprenderás

  • Integrar todo lo aprendido en una página personal.
  • Organizar contenido, estilos y enlaces.

Requisitos del proyecto

  • Una página principal con <header>, <main> y <footer>.
  • Incluye una imagen de perfil con alt, una sección "Sobre mí", una lista de habilidades y un formulario de contacto.
  • Aplica estilos CSS básicos.

Ejemplo de estructura

<header><h1>Mi web</h1></header>
<main>
  <section id="sobre-mi">...</section>
  <section id="portfolio">...</section>
</main>
<footer>Contacto & enlaces</footer>

Entrega

Prepara una carpeta con todos los archivos (HTML, CSS e imágenes). Sube la carpeta comprimida o comparte un enlace si la alojas en GitHub/GitLab.
