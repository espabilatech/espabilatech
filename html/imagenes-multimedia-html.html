Cabeceras o titulares

La etiqueta <h1> hasta <h6> se utilizan para crear diferentes titulares de una página web. Aunque realmente nunca he visto utilizar h5 o h6 en un proyecto real.

El h1 es el título más importante y debería ser único en cada página. Los demás se utilizan para estructurar el contenido en secciones y subsecciones.

Parrafos y enriquecimiento

Para crear un parrafo se utiliza la etiqueta <p>. Por defecto añade margenes arriba y abajo para separar los parrafos entre si.

<b> resalta el texto en negrita.

<u> subraya el texto.

<s> tacha el texto.

<i> letra cursiva.

Estas etiquetas se pueden combinar entre si.
Estas etiquetas no son semánticas, es decir, no aportan significado al texto, solo lo enriquecen visualmente.
