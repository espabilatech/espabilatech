Etiquetas contenedoras en HTML

En HTML, las etiquetas contenedoras (también llamadas etiquetas de bloque o estructurales) son aquellas que sirven para agrupar, organizar o contener otros elementos HTML dentro de ellas, ya sean texto, imágenes u otras etiquetas.

1. Contenedores genéricos

Etiqueta Descripción
<div> Contenedor genérico para agrupar bloques de contenido.
<span> Contenedor en línea para agrupar texto o elementos dentro de una línea.

2. Estructura principal de página

Etiqueta Descripción
<header> Encabezado de una página o sección (logo, menú, etc.)
<nav> Contiene enlaces de navegación.
<main> Contiene el contenido principal del documento.
<section> Agrupa contenido relacionado dentro de una página.
<article> Contiene un contenido independiente, como una noticia o publicación.
<aside> Contenido complementario o lateral (barra lateral).
<footer> Pie de página o sección.

3. Contenedores para contenido específico

Etiqueta Descripción
<form> Contiene elementos de formulario (<input>, <button>, etc.)
<fieldset> Agrupa campos dentro de un formulario.
<label> Asocia texto con un campo de formulario.
<table> Contiene una tabla completa.
<thead>, <tbody>, <tfoot> Subcontenedores dentro de una tabla.
<ul>, <ol> Contienen listas.
<li> Contiene un ítem de lista.
<figure> Agrupa contenido multimedia (imágenes, diagramas, etc.)
<figcaption> Descripción o título para <figure>.
