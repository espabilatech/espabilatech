Día 11 - Git y GitHub: Instalación y uso básico

1. Instalación de Git

  1. Visita https://git-scm.com/ y descarga el instalador para tu sistema operativo.
  2. Ejecuta el instalador y sigue los pasos recomendados.
  3. Verifica la instalación abriendo una terminal y ejecutando: git --version

2. Crear una cuenta en GitHub

  1. Accede a https://github.com/
  2. Haz clic en Sign up y sigue los pasos para crear tu cuenta.

3. Crear un repositorio en GitHub

  1. Inicia sesión en GitHub.
  2. Haz clic en el botón New o Nuevo repositorio.
  3. Asigna un nombre al repositorio, elige si será público o privado y haz clic en Create repository.

4. Clonar el repositorio con GitHub Desktop

  1. Descarga e instala GitHub Desktop.
  2. Inicia sesión con tu cuenta de GitHub.
  3. En la web de GitHub, entra a tu repositorio y haz clic en Code > Open with GitHub Desktop.
  4. Sigue las instrucciones para elegir la carpeta de destino y clonar el repositorio.
