Retros de Programación

¿Qué son los retros?

Los retros de programación son sesiones de revisión y análisis de código donde examinamos ejercicios, proyectos y problemas comunes en el desarrollo web. Estas sesiones nos permiten aprender de forma práctica, identificando patrones, buenas prácticas y errores frecuentes.

En espabilaTech, utilizamos los retros como una herramienta fundamental para reforzar conceptos teóricos con ejemplos prácticos y situaciones reales que te encontrarás en tu carrera como desarrollador frontend.

ℹ️
¿Por qué son importantes?

Los retros te permiten aprender de forma activa, viendo cómo se aplican los conceptos en situaciones reales y cómo resolver problemas comunes en el desarrollo web.

Cómo usar esta documentación

Esta documentación está estructurada de forma similar a GitBook, con un menú lateral para navegar entre secciones y una tabla de contenidos a la derecha para moverte dentro de la página actual.

Variables y tipos

Las variables son contenedores para almacenar datos en nuestros programas. En JavaScript, podemos declarar variables usando let, const o var (aunque este último se considera obsoleto en código moderno).

Declaración de variables

JavaScript 
let nombre = 'espabilaTech';
const edad = 2;
let estaActivo = true;

Tipos de datos

JavaScript tiene varios tipos de datos primitivos:

  • String: Cadenas de texto
  • Number: Números (enteros y decimales)
  • Boolean: Valores lógicos (true/false)
  • Undefined: Variable declarada pero sin valor asignado
  • Null: Valor nulo (ausencia intencional de valor)
  • Symbol: Valor único e inmutable
  • BigInt: Números enteros de precisión arbitraria
⚠️
Cuidado con la coerción de tipos

JavaScript es un lenguaje de tipado dinámico, lo que significa que una variable puede cambiar de tipo durante la ejecución del programa. Esto puede llevar a comportamientos inesperados si no se tiene cuidado.

Operadores

Los operadores nos permiten realizar operaciones con variables y valores. JavaScript incluye operadores aritméticos, de asignación, de comparación, lógicos y más.

Operadores aritméticos

JavaScript 
let a = 10;
let b = 5;

let suma = a + b;      // 15
let resta = a - b;     // 5
let multiplicacion = a * b;  // 50
let division = a / b;  // 2
let modulo = a % b;    // 0
let exponente = a ** b; // 100000

Operadores de comparación

JavaScript 
let x = 5;
let y = '5';

console.log(x == y);   // true (compara valor, con coerción de tipos)
console.log(x === y);  // false (compara valor y tipo)
console.log(x != y);   // false
console.log(x !== y);  // true
console.log(x > 3);    // true
console.log(x <= 5);   // true

Condicionales

Las estructuras condicionales nos permiten ejecutar diferentes bloques de código dependiendo de si una condición se cumple o no.

if, else if, else

JavaScript 
let edad = 18;

if (edad < 18) {
  console.log('Eres menor de edad');
} else if (edad === 18) {
  console.log('Acabas de cumplir la mayoría de edad');
} else {
  console.log('Eres mayor de edad');
}

Operador ternario

JavaScript 
let edad = 20;
let mensaje = edad >= 18 ? 'Eres mayor de edad' : 'Eres menor de edad';

console.log(mensaje); // 'Eres mayor de edad'

switch

JavaScript 
let dia = 3;
let nombreDia;

switch (dia) {
  case 1:
    nombreDia = 'Lunes';
    break;
  case 2:
    nombreDia = 'Martes';
    break;
  case 3:
    nombreDia = 'Miércoles';
    break;
  case 4:
    nombreDia = 'Jueves';
    break;
  case 5:
    nombreDia = 'Viernes';
    break;
  case 6:
    nombreDia = 'Sábado';
    break;
  case 7:
    nombreDia = 'Domingo';
    break;
  default:
    nombreDia = 'Día inválido';
}

console.log(nombreDia); // 'Miércoles'

Bucles

Los bucles nos permiten ejecutar un bloque de código repetidamente mientras se cumpla una condición.

for

JavaScript 
for (let i = 0; i < 5; i++) {
  console.log(`Iteración ${i}`);
}

while

JavaScript 
let i = 0;
while (i < 5) {
  console.log(`Iteración ${i}`);
  i++;
}

do...while

JavaScript 
let i = 0;
do {
  console.log(`Iteración ${i}`);
  i++;
} while (i < 5);

for...of

JavaScript 
const frutas = ['manzana', 'banana', 'naranja', 'uva'];

for (const fruta of frutas) {
  console.log(fruta);
}

Ejercicio 1: Calculadora

Ejercicio práctico

Calculadora básica

Crea una calculadora básica que pueda realizar operaciones de suma, resta, multiplicación y división.

Pasos:

  1. Crea la estructura HTML con botones para números y operaciones.
  2. Estiliza la calculadora con CSS para que tenga un aspecto profesional.
  3. Implementa la lógica en JavaScript para realizar las operaciones.
  4. Añade validación para evitar errores (como división por cero).
Pista: Puedes usar el evento click para capturar cuando el usuario presiona un botón, y el método eval() para evaluar expresiones matemáticas (aunque en aplicaciones reales, es mejor evitar eval() por razones de seguridad).

    <button class='btn' data-valor='9'>9</button>

JavaScript 
document.addEventListener('DOMContentLoaded', () => {
  const display = document.querySelector('.display');
  const botones = document.querySelectorAll('.btn');
  
  let valorActual = '0';
  let operacionPendiente = false;
  
  // Función para actualizar el display
  const actualizarDisplay = () => {
    display.textContent = valorActual;
  };
  
  // Manejar clics en los botones
  botones.forEach(boton => {
    boton.addEventListener('click', () => {
      const valor = boton.getAttribute('data-valor');
      
      // Limpiar
      if (valor === 'C') {
        valorActual = '0';
        operacionPendiente = false;
      }
      // Igual
      else if (valor === '=') {
        try {
          // Evitar eval() en aplicaciones reales
          valorActual = String(eval(valorActual));
          operacionPendiente = false;
        } catch (error) {
          valorActual = 'Error';
        }
      }
      // Operadores
      else if (['+', '-', '*', '/'].includes(valor)) {
        // Evitar operadores consecutivos
        if (operacionPendiente) {
          valorActual = valorActual.slice(0, -1) + valor;
        } else {
          valorActual += valor;
          operacionPendiente = true;
        }
      }
      // Números y punto decimal
      else {
        if (valorActual === '0' || valorActual === 'Error') {
          valorActual = valor;
        } else {
          valorActual += valor;
        }
        operacionPendiente = false;
      }
      
      actualizarDisplay();
    });
  });
});
