Qué aprenderás

  • Qué es la web y cómo funciona.
  • Qué son HTML, CSS y JavaScript y cómo se relacionan.

Objetivo: Crear tu primer archivo index.html y visualizarlo en el navegador.

Estructura mínima de una página HTML

<!DOCTYPE html>
<html lang="es">
  <head>
    <meta charset="UTF-8">
    <meta name="description" content="Mi primera página web">
    <title>Mi primera web</title>
  </head>
  <body>
    <h1>¡Hola mundo!</h1>
    <p>Esta es mi primera página en HTML.</p>
  </body>
</html>

Ejercicio

  1. Crea un archivo llamado index.html y copia el código anterior.
  2. Ábrelo con tu navegador (doble clic o arrastra al navegador).
  3. Modifica el texto del <h1> y del <p>.
