Día 8 - Introducción a CSS (estilos básicos)

Inicio Anterior Siguiente

Qué aprenderás

  • Cómo añadir CSS (inline, internal, external).
  • Selectores básicos, color, tipografía y cajas.

Ejemplo: CSS interno

<style>
  body { font-family: Arial, sans-serif; }
  h1 { color: #2b6cb0; }
  .card { padding:12px; border-radius:6px; }
</style>

Ejercicio

Aplica estilos a una de tus páginas anteriores: cambia fuente, tamaño de título y añade margen a los párrafos. Usa un archivo styles.css externo o un bloque <style>.
