Layers y Compositing

¿Qué son las layers?

El navegador puede crear capas (layers) para aislar partes de la página y optimizar el renderizado, especialmente con position: fixed, transform, will-change o animaciones.

¿Cuándo se crean layers?

  • Elementos con position: fixed o sticky
  • Elementos con transform distinto de none
  • Elementos con will-change
  • Elementos con filter, opacity, canvas, video, etc.

Impacto en el rendimiento

  • Las layers pueden mejorar animaciones y aislar repaints.
  • Demasiadas layers consumen memoria y pueden ralentizar el renderizado.
