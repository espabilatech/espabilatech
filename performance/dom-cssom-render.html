DOM, CSSOM y Renderizado

¿Cómo funciona el navegador?

El navegador convierte el HTML en el DOM y el CSS en el CSSOM. Ambos se combinan para crear el árbol de renderizado, que determina qué se muestra en pantalla.

Fases del renderizado

  1. Parseo HTML → DOM
  2. Parseo CSS → CSSOM
  3. Render Tree: combinación DOM + CSSOM
  4. Layout (Reflow): cálculo de posiciones y tamaños
  5. Paint: dibujo de píxeles
  6. Composite: composición de capas
